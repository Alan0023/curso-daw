"Layout" en CSS

El layout es la disposicion o la forma en la que los elementos se muestran en la pantalla según su tipo.

Display

Es la propiedad más importante de CSS para controlar el alineamiento.

La propiedad displayespecifica la forma en la que se muestra un elemento.

En HTML cada elemento posee un valor por defecto dependiendo del tipo de elemento que sea. La mayoría de los elementos poseen una propiedad blocko inline por defecto.
