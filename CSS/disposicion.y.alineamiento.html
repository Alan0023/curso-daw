"Layout" en CSS

El layout es la disposicion o la forma en la que los elementos se muestran en la pantalla según su tipo.

Display

Es la propiedad más importante de CSS para controlar el alineamiento.

La propiedad displayespecifica la forma en la que se muestra un elemento.

En HTML cada elemento posee un valor por defecto dependiendo del tipo de elemento que sea. La mayoría de los elementos poseen una propiedad blocko inline por defecto.

Elementos de bloque

Los elementos de bloque siempre empiezan en una nueva línea y toman todo el ancho disponible.

Este elemento div es de tipo bloque.

Algunos elementos de bloque:

  • div
  • h1 - h6
  • p
  • form
  • header
  • section
  • footer

Elementos en línea(Inline)

Estos elementos no empiezan en una nueva línea y ocupan sólo el ancho necesario.

Esto es un span dentro un párrafo

Algunos de estos elementos:

  • span
  • a
  • img

Diferencias entre display none; y cisibility: hidden;

Al ocultar un elemento con display: none; la página se mostrará como si el elemento no estuviera ahí.

Al ocultar un elemento con visibility: hidden el elemento no se mostrará pero el espacio que ocupa se quedará en blanco.

display: none;

visibility: hidden;
