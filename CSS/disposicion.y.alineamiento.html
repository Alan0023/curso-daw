Como hemos mencionado anteriormente, un elemento de bloque siempre ocupa todo el ancho disponible.

Asignar una anchura con width evitará que un elemento de bloque se estire a los bordes de su contenedor. La propiedad position

Esta propiedad define el metodo de posicionamiento de un elemento.

Hay 5 tipos de psosición.

  • static
  • relative
  • fixed
  • absolute
  • sticky
    •

Los elementos posicionan con las propiedades top, left, right, bottom. Sin embargo estas propiedades no funcionarán sin especificar primero la position. Además, dependiendo del tipo de position las propiedades funcionarán de una manera distinta.

position: static;

Todos los elementos tienen este valor por defecto.

Las propiedades, top, bottom, left, right no afectan a los elementos con posición estática.

position: relative;

En elemento con este valor se posicionará de manera relativa a su posición normal. Es decir, el centro del eje será sobre si mismo.

Este div tiene position: relative;

position:fixed;

Un elemento con este valor, se posicionará relativamente respecto a la pantalla. Lo que quiere decir que tendrá siempre la misma posición incluso si scrolleamos a la web.

position: absolute;

Un elemento con este valor se posicionará respecto a su contenedor posicionado(no estático).

Si un elemento absoluto no tiene ancestro pocisionados, usará el body y actuará como un position: fixed;.

Este elemento div tiene un position: relative;
Este elemento div tiene una position: absolute;

position: sticky;

Un elemento stiky alternará su comportamiento entre relative y fixed dependiendo de la posición de scroll de la pantalla.
