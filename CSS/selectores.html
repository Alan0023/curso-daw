Selectores de CSS

Los Selectores tienen la funcion de encontrar o seleccionar Elementos de HTML a los que queremos dar estilo

  • Selectores sencillos - Apuntan a elementos básandose en su clase, ., id # o nombre h1.
  • Combinadores - Selectores basados en una específica relacion entre los elementos .contenedor div .
  • Pseudo clases - Selectores basados en el estado del elementos button:hoever.
  • Pseudo elementos - Seleccionan una parte especifica de un elementos
  • De atributo - Selecciona elementos básandose en un atributo o en el valor del atributo.

Selectores de nombre de elemento

Seleccionan los elementos básandose en su nombre de elemento.

Ejemplo: Todos los span serán de color morado. 

                span{
                    color: purple;
                }

Selectores de id

Este selector usa el atributo id de un elemento HTML para seleccionar un elemento específico.

La id de un elemento es única en el documento. Por lo tanto, el selector de id # se usa para especificar un único elemento.

Ejemplo: Este párrafo será el único en estar centrado y de color verde limón.

Los selectores pueden especificar elementos dentro de una clase.

El selector universal

El selector universal * selecciona a totos los elementos.

Selector de agrupación

Sirve para agrupar aquellos elementos que tienen las mismas características

Ejemplo: Digamos que queremos darle la misma sobra que sombras a todos los h1 y h2. En vez de ir a todos los elementos h1 y h2 y darles esta clase podemos hacer lo siguiente: 

        
            .sombras{
                box-shadow: 5px 5px 3px -2px;
            }
            h1{
                box-shadow: 5px 5px 3px -2px;
            }
            h2{
                box-shadow: 5px 5px 3px -2px;
            }

Agrupado quedaría así:

            
                .sombras, h1, h2{
                    box-shadow: 5px 5px 3px -2px;
                }
