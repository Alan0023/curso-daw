Usando flexbox

Para alinear elementos usando flexbox. Éstos deben estar detro de un elemento padre (al que vamos a llamar contenedor) con la propiedad display: flex

Los elementos hijos(que esten detro) de un contenedor flex se convertirán, automaticámente, en elementos flexibles.
