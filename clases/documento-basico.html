Formateo del texto en HTML

ELementos destinados específicamente a formatear el texto:

  • b - Texto en negrita.
  • strong - Texto importante.
  • i - Texto en cursiva
  • em - Texto en enfásis
  • mark - Texto marcado.
  • small - Texto tachado
  • ins - Texto insertado
  • sub - Texto subscrito
  • sup - Texto superescrito

Esto es un párrafo. El texto que querramos destacar visualmente lo haremos con negrita. El texto importante con la etiqueta strong

Elementos de citación

  • blockquote - Define una sección citada de otra fuente.
  • q - Define una cita corta.
  • abbr - Define una abraviatura o acrónimo.
  • address - Define la información de contacto del autor del artículo.
  • cite - Define el título de un trabajo creativo. Como un poema, libro o canción.
  • bdo - Se usa para definir la dirección del texto.
