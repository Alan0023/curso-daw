Links o hipervínculos

Los links se encuentran en todo tipo de páginas. Permiten al usuario navegar por los documentos de la web.

Cuando clicas en un link te mueve a otro documento o a una parte del mismo.

Cuando se pulsa el cursor sobre un lisk, éste se torna en uan mano.

Un link se deine mediante el elemento <a>

La sintaxis de un link, por norma general, es : <a href="url">Text del link</a>

El atributo más importante de un link es el "herf" ya que es éste el que indica el punto al que nos llevará el link.

Ejemplos:

Link a google
Link a la teoria de CSS

Los links tienen 3 estados:

  • Son azules cuando no han sido visitados
  • Morados cuando ya han sido clicados.
  • Y rojos cuando están activos.

Los links se pueden modificar mediante CSS para que tengan un aspecto totalmente distinto.
