Id's y clases

Las id y las clases son atributos que se pueden asignar a los elementos de HTML. Mientras que las clases se pueden asignar a varios, elementos, las id son únicas y solo puede tenerla un elemento.

Clases

La principal función de las clases es apuntar a varios elementos mediante CSS o la manipulación mediante JavaScipt a múltiples elementos.
