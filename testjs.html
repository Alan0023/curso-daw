Prueba de JavaScript

Si el HTML proporciona la estructura del documento, el CSS el estilo visul, entonces es JS(JavaScript) el que proporciona la interactividad de la web con el usuario.

Con JS podemos hacer que la página reaccione ante las distintas acciones del usuario. Hacer clic, pasar el ratón sobre el objeto en concreto, hacer scroll , usar el teclado, etc.

Al igual que HTML nops provee de la etiqueta style para insertar el CSS. Tenemos la etiqueta script para escribir JavaScript. El código también puede incorporarse como atributo de los elementos (como sucede con el atributo style) por lo que dependera del criterio del desarrollador.
